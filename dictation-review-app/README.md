# 小葵の背默

给家长和初中生使用的纸笔背默复习网页。学生在纸上写，家长在网页里记录正确、错误、拼写错、音标错或具体错字。

## 两种使用方式

### 1. iPad / GitHub Pages 静态版

适合把 `public/` 里的文件上传到 GitHub Pages，然后在 iPad 浏览器打开。

静态版可以直接使用：

- 添加英语单词、中文词语、古诗词或句子。
- 点击“直接保存”，只记录你输入的原始内容。
- 自动进入今日背默任务。
- 记录纸笔背默结果。
- 保存奖励、抵扣和最近 7 天达成记录。
- 导出和导入 JSON 备份。

静态版不会保存 OpenAI API Key，也不会直接联网补全音标、释义、注释和来源。这是为了避免把密钥暴露到浏览器或 GitHub。

### 2. 电脑后端联网版

适合在电脑上运行后端，让电脑负责保管 API Key 和联网补全资料。

```bash
cd dictation-review-app
npm install
npm start
```

本机打开：

```text
http://localhost:3000
```

iPad 在同一个局域网里访问电脑 IP，例如：

```text
http://电脑局域网IP:3000
```

`.env` 示例：

```bash
PORT=3000
OPENAI_API_KEY=your_api_key
OPENAI_MODEL=gpt-5-mini
```

电脑后端版可以使用“联网查询”，也可以在学习库里对静态版先保存的内容点击“联网补全”。

## 添加内容

- 静态版：输入后点“直接保存”，内容马上进入学习库和今日任务。
- 静态版高级入口：点“生成填写卡”，可以手动修改 JSON 后保存。
- 后端版：点“联网查询”，确认资料后保存。
- 已经保存的内容可以在“学习库”里点击“联网补全”更新完整资料。

## 数据备份

导出：

1. 页面底部点击“导出备份”。
2. 保存下载的 JSON 文件。

导入：

1. 页面底部点击“导入备份”。
2. 选择之前导出的 JSON 文件。
3. 程序会先校验结构，破损文件不会覆盖当前数据。

## 复习规则

常规正确后的复习间隔：

| 连续正确次数 | 下次复习 |
| --- | --- |
| 1 | 1 天后 |
| 2 | 3 天后 |
| 3 | 7 天后 |
| 4 | 15 天后 |
| 5 | 30 天后 |
| 6 | 60 天后并进入熟练 |

错误后：

- 当前维度连续正确次数清零。
- 第二天重新复习。
- 当天会进入重背队列。
- 同日重背写对不增加长期连续正确次数。
- 漏做或逾期不惩罚，只要到期日不晚于今天就进入今日任务。
